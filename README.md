### 🎮 全球扫雷 (Global Minesweeper)

基于 SwiftUI 开发的高性能原生扫雷，专为硬核玩家打造。

**项目核心特色：**
* **80+ 语言支持**：内置包括中文（简体/繁体/香港）、英语、俄语等海量语言包，结算界面全屏适配所选语言。
* **极致操作手感**：支持 **0.25秒极速标红** 标记，取消延迟感；具备 **首点绝对安全** 逻辑，开局第一点绝不踩雷。
* **多级挑战维度**：提供简单、中等、困难及 **25x25 (50雷)** 的特别困难模式，矩阵画面智能适配缩放。
* **轻量原生体验**：代码精简高效，运行内存占用极低，带给您最丝滑的 macOS 游戏交互。
